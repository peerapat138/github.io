Arduino Heartbeatจตุเทวดา พิทักษา อัตราชีพ
บทคัดย่อ (Abstract) โครงการนี้ออกแบบอุปกรณ์ IoT สำหรับตรวจวัดอัตราการเต้นของหัวใจแบบเรียลไทม์ เพื่อสนับสนุนการดูแลสุขภาพหัวใจ โดยเฉพาะผู้ป่วยโรคหัวใจหรือผู้ที่ต้องการเฝ้าระวังสุขภาพอย่างต่อเนื่อง อุปกรณ์ใช้ Pulse Sensor ซึ่งติดตั้งบนไมโครคอนโทรลเลอร์ที่เชื่อมต่อกับ Wi-Fi เพื่อวัดค่าชีพจร (BPM) และส่งข้อมูลแบบเรียลไทม์ไปยัง LINE chat bot เมื่อพบชีพจรผิดปกติ ระบบจะส่งการแจ้งเตือนทันทีผ่าน LINE ทำให้ผู้ใช้สามารถดำเนินการได้อย่างรวดเร็ว นอกจากนี้ ค่าชีพจรจะถูกแสดงบนจอ LCD สำหรับการตรวจสอบเบื้องต้นและเก็บบันทึกลงใน LINE chat bot เพื่อให้เข้าถึงข้อมูลย้อนหลังได้ ผู้ใช้สามารถตั้งค่าเปิด-ปิดการใช้งานตามความต้องการ อุปกรณ์นี้ถูกออกแบบให้มีราคาไม่สูงและสามารถพัฒนาต่อยอดได้ง่าย เหมาะสำหรับผู้ใช้ทั่วไปและผู้ป่วยที่ต้องการตรวจสอบอัตราการเต้นของหัวใจอย่างสม่ำเสมอ
